## Scaffolding Your First Mango Project

With NPM:

```bash
npm create @mango-js my-app
```

With Yarn:

```bash
yarn create @mango-js my-app
```

With PNPM:

```bash
pnpm create @mango-js my-app
```

## Running Project in Development Mode

```bash
cd my-app
npm run start
```
